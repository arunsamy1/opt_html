e-dumper One Liner Q

A simple one-at-a-time text viewer for reviewing quotes, proverbs, or any line-by-line content.

Quick Start

  1. Open e-dumper-oneline-q.html in any modern web browser
  2. Import your text file using the "Import Text" button
  3. Navigate through lines with Previous/Next or arrow keys

Features

Display

  • Shows one line at a time for focused reading
  • Large, easy-to-read text display
  • Progress indicator (X of Y lines)

Organization

  • Shuffle lines to randomize order
  • Navigate forward and backward through content
  • Jump to specific lines (optional enhancement)

Use Cases

  • Memorizing quotes and proverbs
  • Reviewing interview questions one at a time
  • Studying one fact at a time
  • Reading through any multi-line content sequentially

File Format

Text files should be plain text (.txt) with one item per line:

The early bird catches the worm.
A stitch in time saves nine.
Actions speak louder than words.
Better late than never.
Birds of a feather flock together.

Format Rules

  • One line per item
  • Lines are trimmed of leading/trailing whitespace
  • Empty lines are ignored

Keys Action
Left Arrow Previous line

Data Persistence

Your progress is automatically saved to localStorage:

  • Current line position
  • Loaded lines

Click "Clear Progress" in the sidebar to reset all data.

Running the App

Simply open e-dumper-oneline-q.html in a browser. No server required.

# Using Python
python -m http.server 8000 -d e-dumper/

# Using npx
npx serve e-dumper/

Then open http://localhost:8000/e-dumper-oneline-q.html

Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
