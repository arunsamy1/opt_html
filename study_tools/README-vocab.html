e-dumper Vocab

A standalone, offline-capable flashcard app for vocabulary learning. Runs entirely in the browser — no server, no account, no installation required.

Quick Start

  1. Open e-dumper-vocab.html in any modern web browser
  2. Click the ☰ toggle to open the sidebar
  3. Click 📂 Import Words and select a .csv or .txt file
  4. Cards load automatically — the sidebar collapses and study begins
  5. Press Show Meaning (or ↑) to reveal the definition, then use Next → (or →) to advance

Sidebar

The sidebar is collapsed by default. Click ☰ to open it and ✕ to close it. On mobile it is pinned to the top of the screen.

Flashcard Mode

The default study mode. Each card shows the word (or meaning in Swap mode) and hides the answer until you choose to reveal it.

Navigation bar

ButtonAction
← PreviousGo to the previous card.
Show MeaningReveal the hidden answer. Button label changes to Meaning Shown. The revealed text is highlighted in green.
⚑ FlagToggle a flag on the current word. Flagged cards show an amber FLAGGED badge.
# / GoType a card number and press Enter or click Go to jump directly to it.
Next →Go to the next card.

Progress bar

Displays the current position (Word X of Y) and completion percentage. The bar fills from red to green as you advance through the deck.

Tip: Shuffle and Swap work together — you can randomise the order while also practising meaning-to-word recognition.

Quiz Mode

Enable 🎯 Quiz Mode in the sidebar to switch from passive flashcards to active multiple-choice testing.

How it works

  • Each card shows the word (or meaning in Swap mode) along with up to 4 answer choices — one correct answer plus up to three random distractors drawn from the rest of the list.
  • Tap or click an option to lock in your answer.
  • The correct choice turns green. A wrong choice turns red and the correct answer is highlighted simultaneously.
  • All options are disabled after answering — you cannot change your selection.
  • Next → is locked until the current card is answered.
  • A live score badge (🎯 correct / total) appears in the progress bar row.

Ending a quiz

The quiz ends automatically after the last card is answered. You can also click ⏹ End Quiz (visible in the navigation bar during quiz mode) at any time to jump straight to the results.

Quiz summary

The summary screen shows your score, percentage correct, and a performance message.

ButtonAction
🔄 Restart QuizReset the score, return to word 1, and reshuffle if Shuffle is on.
📥 Export Wrong AnswersDownload every word answered incorrectly as quiz_wrong_answers.csv, ready to re-import as a focused study list. This button is hidden when the score is perfect.
Exit Quiz ModeReturn to normal flashcard mode at word 1.
Note: Wrong answers are tracked per quiz session only. Restarting or exiting the quiz clears the wrong-answer list.

Quiz mode interactions with other settings

  • Shuffle — if Shuffle is on when Quiz Mode is enabled, the deck is reshuffled immediately.
  • Swap — Quiz Mode respects Swap mode; the options will be words when Swap is active.
  • Flag — you can still flag cards mid-quiz. Useful for marking words you found difficult.
  • Keyboard shortcuts — see the Keyboard Shortcuts section for quiz-specific behaviour.

Flagged Words

Mark any card as flagged using the ⚑ Flag button or the M key. Flagged cards display an amber FLAGGED badge in the top-left corner of the card. Pressing the button again removes the flag.

  • The sidebar shows the live count: ⚑ Review Flagged (N).
  • Clicking Review Flagged jumps to the first flagged word. Each subsequent click advances to the next flagged word, wrapping back to the start after the last one.
  • Flags survive shuffling — each word carries its flag regardless of deck order.
  • Flags are saved to localStorage and restored when you reopen the app.
  • Use 📥 Export Flagged to save your flagged words as a CSV file for a focused follow-up session.

File Format

Word list files must be plain text (.txt or .csv) with one entry per line. Each line contains the word and its meaning separated by a comma or pipe character.

apple,a fruit that is red or green
banana,a long yellow fruit
cherry,a small round red fruit

Separator options

SeparatorExample
Comma ,word,meaning
Pipe |word|meaning

Rules

  • Blank lines are ignored.
  • Lines without a separator are skipped.
  • If the meaning itself contains a comma, use the pipe separator to avoid splitting the meaning incorrectly.
  • Leading and trailing whitespace is stripped from both the word and the meaning.
  • Importing a new file replaces the current word list and resets position and all flags.

Data Persistence

Progress is automatically saved to localStorage after every action and restored the next time you open the file in the same browser.

What is saved

  • The full word list, including the flag state of each word
  • Current card position
  • Shuffle and Swap checkbox states

What is not saved

  • Quiz Mode state — Quiz Mode is always off on load
  • Quiz scores and wrong-answer lists — these are session-only and reset each time

Click 🗑️ Clear Progress in the sidebar to erase all saved data and start fresh.

Note: localStorage is scoped to the page origin (protocol + hostname + port). If you open the file from a different folder path or a different local server port, previously saved state will not be available.

Running the App

Double-click e-dumper-vocab.html to open it directly in a browser — no server needed. To serve it locally:

# Python
python -m http.server 8000 -d e-dumper/

# Node / npx
npx serve e-dumper/

Then open http://localhost:8000/e-dumper-vocab.html in your browser.

Browser & Device Support

Works in all modern browsers on desktop and mobile:

  • Chrome / Edge 90+
  • Firefox 90+
  • Safari 14+ (iOS and macOS)

The app is fully responsive. On mobile the sidebar becomes a collapsible panel pinned to the top of the screen. All touch targets meet the 44 px minimum for comfortable tapping, and the 300 ms tap delay is removed.
