How it works: Your file is encrypted with AES-256-GCM entirely in the browser. The password is stretched with PBKDF2 (310,000 iterations, SHA-256). A random salt and IV are generated per encryption and prepended to the output. The encrypted file downloads as filename_enc.ext. To decrypt, select the encrypted file, enter the same password, and click Decrypt & Download — the _enc suffix is removed automatically.
